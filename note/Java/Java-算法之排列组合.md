```java
/*题目概述：有 n 个 a，m 个 b，求排列组合有多少种可能？*/
public int dfs(int n, int m){
    if(n == 0) return 1;//剩下都是 b，排列只有一种可能
    if(m == 0) return 1;//剩下都是 a，排列只有一种可能
    return dfs(n-1,m) + dfs(n,m-1);//第一个选 a，或者第一个选 b
}
```
